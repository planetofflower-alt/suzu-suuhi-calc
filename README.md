# 運命数 計算ツール｜すず

数秘術の運命数（Life Path Number）を生年月日から計算する、純クライアントサイドの静的ツール。

🔗 by [すず｜数秘トリセツ](https://note.com/suzu_suuhi)

## プライバシー

- 入力した生年月日は **ブラウザ内のみで計算** されます。
- サーバー送信・保存・第三者解析（GA等）一切なし。
- 外部APIを呼びません = **APIキー・シークレットは存在しません**。
- 単一HTMLファイル（外部リソース依存ゼロ）。コードは本リポジトリで全文公開。

## 仕様

生年月日の数字を全部足し、1桁になるまで還元。途中で `11 / 22 / 33` が出たらそこで止めて「マスターナンバー」。

```
例: 1995-03-21 → 1+9+9+5+3+2+1 = 30 → 3+0 = 3
例: 1989-12-08 → 1+9+8+9+1+2+8 = 38 → 3+8 = 11 (マスター)
```

## ローカル確認

`index.html` をブラウザで開くだけ。ビルド不要。

## デプロイ

Vercel / GitHub Pages / Netlify など、静的ホスティングならどれでも。設定なしでそのまま動きます。
